Assistant Personnel Intelligent

Description

Application mobile développée avec React Native permettant aux utilisateurs de gérer leurs tâches quotidiennes grâce à un assistant vocal intelligent. L'application combine des techniques de NLP (Natural Language Processing) avec des algorithmes de planification automatique pour offrir une expérience utilisateur fluide.

Fonctionnalités Clés

  • Reconnaissance vocale en temps réel
  • Gestion intelligente des tâches et calendrier
  • Intégration avec les services externes (météo, trafic, etc.)
  • Apprentissage des préférences de l'utilisateur

Technologies Utilisées

React Native, Python, TensorFlow, Firebase
